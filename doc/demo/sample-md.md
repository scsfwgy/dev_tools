# Tools24 开发者工具箱

> 在线开发者工具集，浏览器本地处理，**不上传数据**。

## 功能列表

### 已上线
- **JSON 格式化**：格式化、压缩、树形查看
- **编码转换**：URL / Base64 / Unicode 互转
- **时间戳**：秒/毫秒戳与日期互转

### 规划中
1. 正则表达式测试
2. SQL 格式化
3. 颜色转换

## 使用示例

```bash
# 启动本地开发
npm install
npm run dev
```
